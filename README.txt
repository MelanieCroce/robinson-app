- Installe nodejs

- avec git bash : 
	npm install -g cordova
	cordova create robinson com.robinson.app Robinson
	cordova platform add android --save
	
- fais un cordova requirements et tu me screen ce que ça te donne

- Supprime le dossier www, et fais un git clone dans le dossier :
	git clone https://github.com/MelanieCroce/robinson-app

(tu dois avoir le dossier .git et le www dans la racine du projet)


La doc si besoin : https://cordova.apache.org/docs/en/latest/guide/cli/index.html
